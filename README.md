# 繁星 | AI视频共创平台
## 用群体智慧，在AI时代，做一部属于我们所有人的故事

---

### 🎯 前言
我是一名媒体从业者，亲眼见证了AI视频技术拉平创作门槛后，整个行业正在陷入的困境：
内容爆炸式增长，创作者陷入“酒香也怕巷子深”的流量内卷，哪怕是天才的创意，也很难在海量信息里获得正反馈；
AI逼得所有人被迫走向单兵作战，要同时兼顾脑洞、制作、运营，无数擅长执行、深耕细节的从业者被淘汰，陷入前所未有的焦虑；
更让我不安的是，当所有人都执着于做“自己的专属内容”，人与人正在变成一座座没有交集的孤岛，我们失去了靠共同故事连接彼此的能力。

我做「繁星」的初衷，就是想把这股无序狂奔的生产力，变得有序。
我想打造一个街坊式的共创社区，让AI成为拓宽人力边界的工具，而非制造焦虑的洪水猛兽；让擅长脑洞的人开个头，擅长细节的人接下去，擅长执行的人做收尾，真正做到人尽其用；让观众也能成为内容的主人，用投票共同孕育一部属于所有人的作品。

就像《人类的群星闪耀时》里写的那样，个体的光芒或许微弱，但无数普通人的智慧聚在一起，就能变成照亮彼此的繁星。这就是这个项目的愿景：在AI时代，造一片专属于人类群体智慧绽放、彼此共振的乐园。

---

### ✨ 项目核心亮点
- ✅ **零门槛全民共创**：不设时长、工具、经验限制，会开脑洞、会用AI生图、会剪辑配音，甚至只会投票提建议，都能参与
- ✅ **观众全程深度参与**：从脑洞选题到角色设定，全流程由观众票选决定，共同养成属于自己的“玫瑰”，建立天然的情感羁绊
- ✅ **人尽其用，拒绝内卷**：不用逼自己做全能创作者，你只需要负责自己擅长的环节，剩下的交给同频的伙伴
- ✅ **收益全透明共享**：项目总流水70%全额分给共创者，规则全公开，流水全公示，知识产权全员共有
- ✅ **天然成熟的商业闭环**：共创内容天然适配互动影游、多元宇宙叙事，影视改编、商单定制等变现路径已被市场充分验证

---

### 🤝 谁能参与？
无论你是哪一类人，都能在这里找到自己的位置：
- 🧠 **脑洞创作者**：擅长写剧本、出点子、定剧情走向，负责项目的灵魂内核
- 🎨 **AI视觉创作者**：擅长AI生图、分镜制作、视频生成，负责把脑洞变成画面
- 🎬 **后期创作者**：擅长剪辑、配音、配乐、调色，负责内容的最终落地
- 👥 **观众共创者**：哪怕没有创作经验，也能投票定剧情、提建议，决定故事的走向，同样能参与收益分红

<details>

1.  **脑洞征集**：所有人都可以提交故事创意、剧情脑洞，不限题材、不限风格
2.  **全民票选**：观众投票选出最感兴趣的脑洞，确定项目的核心主线
3.  **角色共创**：擅长视觉创作的伙伴提交角色设定，再次由全民票选最终人设
4.  **内容接力**：不设最低时长、不限工具，所有人都可以提交对应剧情的视频片段，票选最优内容进入正片
5.  **成片整合**：擅长执行、细节把控的伙伴，完成全片风格统一、配音配乐、剪辑整合，最终成片全平台发布
6.  **收益分红**：项目所有变现收益，按贡献度全透明分给每一位参与者，观众投票贡献同样计入权重

</details>

### 💰 收益规则与商业模式
<details>
<summary>👉 点这里查看详细规则</summary>

#### 收益分配规则
- 项目全平台总流水的**70%全额纳入共创者分红池**，剩余30%用于项目宣发、合规运营、服务器成本
- 分红按贡献度权重分配：主线剧情创作40%、AI视觉创作30%、后期整合20%、观众投票/社区运营10%
- 每月5号前，全平台公示上月流水明细、运营成本、分红明细，每一笔收支全透明可查
- 核心共创者永久享有项目IP后续衍生收益（影视改编、游戏、周边、商单等）的分红权

#### 成熟变现路径
1.  **互动影游付费**：前3集免费引流，后续主线剧情、隐藏分支、专属结局采用低客单价付费解锁，已被红果短剧、番茄影游验证百亿级市场规模
2.  **全平台流量分账**：成片在B站、抖音、西瓜视频等全平台发布，获取官方流量分成、商单广告收益
3.  **IP衍生变现**：成熟的故事IP可进行影视改编、短剧开发、游戏授权、周边衍生等长线变现
4.  **品牌定制共创**：为品牌方定制专属共创内容，实现商业变现

</details>
